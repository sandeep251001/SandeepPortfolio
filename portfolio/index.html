Project 3

Gender Classification By Voice

The project "Gender Classification By Voice" aims to develop a machine learning system that can accurately predict the gender of individuals based on their voice characteristics. This project utilizes a numerical dataset containing voice-related features and applies several classification algorithms, including Support Vector Machines (SVM), Random Forest (RF), Decision Trees (DT), Naive Bayes, and a combination of SVM and RF, to achieve accurate gender classification.
